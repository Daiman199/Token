Cara Install Bot Token Di Termux :
------
- `pkg install nodejs -y`
- `pkg install git -y`
- `git clone https://github.com/Gendjex/Token`
- `pkg install nano`
- `cd AlphatTJ`
- `npm install`
- `npm start`
